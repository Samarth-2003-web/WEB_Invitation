# 💍 Engagement Celebration Website

An elegant, modern engagement celebration website for Saurabh P Shetty & Poornima D Shetty, celebrating their engagement on **14 May 2026**.

## ✨ Features

### Design & Animation
- **Custom Cursor Ring** - Elegant animated cursor that follows mouse movement
- **Starfield Animation** - Twinkling stars canvas background
- **Falling Petals** - Animated rose petals falling on hero section
- **3D Rotating Ring** - Interactive 3D ring animation with gradient
- **Scroll Animations** - Smooth fade-up reveals on scroll
- **Card Hover Effects** - Interactive card lifts with glow

### Interactive Elements
- **Live Countdown Timer** - Real-time countdown updating every second
- **Smooth Scrolling** - Elegant scroll-behavior throughout
- **Mouse Parallax** - Hero content responds to mouse movement
- **Google Maps Integration** - Embedded venue location

### Glassmorphism Effects
- **Glass Morphism UI** - Semi-transparent frosted glass backgrounds with backdrop blur
- **Elevated Cards** - Glass cards with shadow and glow effects
- **Layered Backgrounds** - Depth through glass morphism on all sections

## 🔧 How to Use

1. **Open in Browser**: Simply open the HTML file in any modern web browser
2. **No Installation Required**: This is a static website with no dependencies
3. **Offline Support**: Works offline except for Google Maps

### Customization

Edit the HTML file to customize:

```html
<!-- Change couple names -->
<div class="hero-names">Your Name<br>&amp;<em>Partner Name</em></div>

<!-- Change date -->
<div class="hero-date">Your Date</div>

<!-- Change venue -->
<div class="card-value">Your Venue</div>
```

### Color Customization

Edit CSS variables in the `<style>` section:

```css
:root {
    --gold: #00d4ff;      /* Primary color */
    --rose: #ff006e;      /* Accent color */
    --deep: #0a0e27;      /* Background color */
    --cream: #e0e0e0;     /* Text color */
}
```
